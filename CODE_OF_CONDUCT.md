# Code Of Conduct

Webhookery uses the Contributor Covenant Code of Conduct, version 2.1, as the
baseline for public project participation:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

Project-specific reporting route:

- Security-sensitive reports: follow `SECURITY.md`.
- Conduct or moderation concerns: contact the maintainer through the private
  channel listed in `SECURITY.md`.

Do not include API keys, webhook secrets, bearer tokens, raw provider
signatures, raw payload bodies, private keys, customer data, or exploit
payloads in public issues, pull requests, discussions, or examples.
