# External Review Findings Template

Use this template to track findings from an external security or production-
maturity review. Keep sensitive proof material outside public source control.
Do not include secrets, raw payload bodies, customer data, exploit payloads,
private keys, bearer/session tokens, provider credentials, raw signatures, or
database URLs with real credentials.
